========================================

الملفات المطلوبة في مشروع Apps Script:

1. Code.gs
   - نوع الملف: Script (.gs)
   - الوصف: الكود الخادم الرئيسي
   - يحتوي على: جميع الدوال والمنطق

2. login.html  
   - نوع الملف: HTML
   - الوصف: صفحة تسجيل الدخول
   - يحتوي على: نموذج تسجيل الدخول + CSS + JavaScript

3. dashboard.html
   - نوع الملف: HTML  
   - الوصف: لوحة التحكم الرئيسية
   - يحتوي على: واجهة البحث والإدارة + CSS + JavaScript

4. admin.html
   - نوع الملف: HTML
   - الوصف: صفحة إدارة الادمن
   - يحتوي على: نموذج إضافة مديرين + CSS + JavaScript

========================================

خطوات سريعة:
1. انشئ مشروع جديد في script.google.com
2. احذف Code.gs الافتراضي
3. أضف الملفات الأربعة أعلاه
4. غيّر SPREADSHEET_ID في Code.gs
5. انشر كتطبيق ويب
